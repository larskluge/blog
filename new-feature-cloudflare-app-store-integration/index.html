New feature: CloudFlare App Store Integration

Aug 7th, 2014

Over the past few weeks, we’ve been busy building a new tool to allow businesses to set up free bug bounty programs in just one click. Today, we’re excited to share the news that CrowdCurity is joining a powerful group of apps that are part of the CloudFlare App Store.

To set up a bug bounty program through Cloudflare, all you need to do is login and add the CrowdCurity app from the CloudFlare App Store. Once you have entered details about the scope and focus of the free security test you would like to run on your site, your program will be created and open to our platform of over 1200 security testers.

CloudFlare is a leader in providing performance and security for over 2 million websites around the world. (We’ve happily used CloudFlare to secure CrowdCurity.) Its platform serves as a content delivery network that shields sites from DDoS attacks, speeds up web traffic, and provides invaluable protection to some of the most critical sites on the web. In addition to protecting its users from malicious attacks, CloudFlare has helped secure the web by making it easier for new tools to be deployed without requiring changes in site code.

We’re proud to be in great company in the CloudFlare App Store and to connect businesses to security researchers who will make their sites more secure.

Are you a CloudFlare user? Add the CrowdCurity app to your dashboard, and create a free bug bounty program today!
