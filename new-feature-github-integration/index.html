New feature: Github integration

Jul 15th, 2014

Recently, we added Github integration to CrowdCurity to help businesses better manage vulnerabilities reported through our platform. As Github users ourselves, we saw an opportunity to build a feature that would make the programs we use to build and secure our site work better together. For businesses using Github, this integration will save time and reduce the amount of work it takes to patch a reported vulnerability.

Integrating CrowdCurity with your Github account is simple. To get started: 1— Login to your CrowdCurity account and navigate to your Program Settings in the Dashboard. 2— Once you’ve opened program settings, click on “Integrations” at the bottom of the menu on the left of the screen.

3— Enter a token and repo or username to link your account, and select Integrate.

From this point on, you will have the option to send vulnerability reports straight to Github from the reports dashboard. If for any reason you need to generate a new token, just follow the steps above to reconnect your Github account.

This integration improved how we handle our own vulnerability management process by removing time-consuming steps between receiving a vulnerability report to patching it. Is there a development tool that you would like to be integrated with CrowdCurity? Tell us on Twitter!
